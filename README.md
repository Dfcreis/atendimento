# AirHelpDesk

Sistema de atendimento aéreo automatizado, desenvolvido para fins educacionais.

## Sobre
Este projeto foi inspirado em rotinas de atendimento de companhias aéreas, mas *não possui vínculo com nenhuma empresa real*. Ele permite gerar protocolos, registrar solicitações e simular isenção de taxas (Waive), cancelamentos e outros processos de SAC.

## Tecnologias
- HTML
- CSS
- JavaScript

## Uso
1. Abra o index.html no navegador.
2. Preencha os campos do formulário.
3. Clique em "Gerar Resultado" para ver o protocolo.

## Aviso
⚠ Este projeto é *educacional e autoral. Qualquer semelhança com sistemas reais é **apenas inspiracional*.
